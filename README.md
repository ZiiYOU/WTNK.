## 📌 주요 기능

- JWT 인증 서버를 활용한 로그인 및 회원가입 구현
- JSON Placeholder api 를 활용한 게시글 viewer 구현
- JWT 토큰을 이용한 인증/인가 구현 및 리다이렉트 처리
- tailwind 를 이용한 스타일링
- zustand를 통한 client-side 전역 state 관리
- axios, tanstack-query를 통한 외부통신 및 server-side 전역 state 관리
